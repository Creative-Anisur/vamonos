Karatsuba's Multiplication Algorithm

if x < 10 or y < 10 return x * y deg = ⌊ max(length(p), length(q)) / 2 ⌋ p-high, p-low = split p as a string at deg q-high, q-low = split q as a string at deg a = karatsuba(p-high, q-high) b = karatsuba(p-high + p-low, q-high + q-low) c = karatsuba(p-low, q-low) result = c * 10^(2 * deg) + (b - c - a) * 10^deg + a return result
